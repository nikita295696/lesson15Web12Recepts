Сайт с рецептами.

Рецепт имеет поля:
    - уникальный идентификатор
    - название
    - инструкция (описание)

1. Добавить страницу рецепты с формой с возможностью создавать/редактировать/удалять рецепты. (form.php)
2. Реализовать страницу, которая будет возвращать случайный рецепт в виде JSON. (api.php)
3. Реализовать страницу для получения через JS случайного рецепта и отобразить его на главной странице.  (index.html)
Добавить возможность сохранить или пропустить случайный рецепт.

Добавить задание на github. В проекте должны присутствовать 2 ветки - main (главная), dev (для разработки).
Домашнее задание скинуть в виде ссылки на github
